# Changelog - KonBot v2.2.1

## Исправления

### Критические ошибки
- Исправлена проверка владельца временного канала (исключения при отсутствии voice.channel)
- Переработана система проверок для команд временных каналов (/lock, /unlock, /vkick)
- Управляющие кнопки теперь отправляются в текстовые каналы вместо голосовых

### Безопасность
- Добавлена система Rate Limiting:
  - Авторизация: 3 попытки за 5 минут
  - Кастомные команды: 5 добавлений за минуту
  - Временные каналы: 2 создания за 5 минут
- Валидация URL для изображений в настройках
- Автоматические backup данных перед сохранением

## Новые функции

### Временные каналы
- Команда /temp_status - просмотр статуса всех активных каналов
- Расширенное логирование действий с каналами
- Проверка лимита каналов в категории (макс. 50)

### Улучшения системы
- Graceful Shutdown - корректное завершение работы
- Улучшена обработка ошибок в команде wipe_dms
- Оптимизированы запросы к Discord API

## Технические улучшения

- Улучшена структура данных для многопользовательской работы
- Расширенное логирование всех операций
- Улучшено кэширование информации

## Требования

- discord.py[voice]>=2.3.0
- aiohttp>=3.8.0
- Python 3.8+

## Примечания

- Все изменения обратно совместимы
- Не требуется миграция данных
- Рекомендуется создать backup перед обновлением
